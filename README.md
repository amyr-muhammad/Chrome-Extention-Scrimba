# Chrome-Extention-Scrimba

## 🚀 Chrome Extension: Lead Tracker
A lightweight, efficient browser extension designed to help users save and manage web URLs instantly. Built as part of the Scrimba Frontend Developer Path, this project focuses on DOM manipulation, local storage, and leveraging the Chrome Extension API.

### 🎯 Features
 One-Click Save: Instantly save the current tab URL using Chrome.tabs API.

 Manual Input: Type in any custom URL or note to save it to your list.

 Data Persistence: Uses localStorage so your leads stay saved even after closing the browser.

 Quick Clear: Double-click the "Delete All" button to wipe the list and start fresh.

### 🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom UI)

Logic: Vanilla JavaScript (ES6+)

APIs: Chrome Extension API (Manifest V3)

Storage: Browser LocalStorage

### 🧠 Lessons Learned With scrimba
While building this project, I strengthened my understanding of:

Asynchronous JavaScript: Handling the chrome.tabs.query method to retrieve active URLs.

Deployment: Converting a standard web app into a functional .json manifest-based extension.

UI/UX: Designing a compact, functional pop-up interface using CSS.
